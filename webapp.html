Project Summary

The aim of this project was to build a simple Terraform project to demonstrate my knowledge of Terraform and AWS. Here are the key specifications:

  • Terraform Version: >= 1.3.3
  • Usage of publicly available Terraform Modules
  • Project can be applied multiple times within a single AWS Region
  • Created Resources:
    • VPC
    • EKS
    • EC2 instance as EKS worker node
    • A container with a web interface
    • Load balancer that exposes the web interface of the container
